
 🌦️ Weather App

A simple and responsive Weather App that shows real-time weather information for any city.

This project was built as a practice project by following a YouTube tutorial, helping me understand how to work with APIs and improve my frontend development skills.


## 🚀 Features

- 🌍 Search weather by city name
- 🌡️ Displays temperature, humidity, and weather conditions
- ⚡ Real-time data using weather API
- 📱 Responsive and clean user interface


## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Weather API (e.g., OpenWeather API)
